# Artemis II Palantir Tracker

GitHub Pages deployment of the Artemis II tactical dashboard.

## URLs

- **Dashboard**: `https://YOUR_USERNAME.github.io/artemis-tracker`
- **API Data**: `https://YOUR_USERNAME.github.io/artemis-tracker/api/data.json`

## Setup

1. Create a new GitHub repository named `artemis-tracker`
2. Push these files to the `main` branch
3. Go to Settings → Pages → Source → Select "main" branch
4. GitHub Actions will auto-update data every 10 minutes

## Data Source

NASA JPL Horizons API - Artemis II Orion (ID: -1024)
